# Informações do pedido

## Objetivo

Esta API permite que você recupere informações detalhadas sobre uma etiqueta de frete específica, utilizando o ID único da etiqueta gerado previamente pela API de Envio de Frete. Ao fazer uma requisição para este endpoint, a API retornará um objeto JSON contendo todos os detalhes associados àquela etiqueta.

O objeto JSON retornado contém os seguintes campos:

`id`: (String) O ID único da etiqueta de frete gerada pela SuperFrete. Este é o identificador principal da etiqueta.

`protocol`: (String) Um identificador de protocolo interno da SuperFrete, geralmente idêntico ao id.

`format`: (String) O formato da embalagem utilizado (ex: "box").

`delivery`: (Integer) O prazo de entrega em dias úteis informado no momento do cálculo do frete.

`delivery_min`: (Integer) O prazo mínimo de entrega em dias úteis informado no momento do cálculo do frete.

`delivery_max`: (Integer) O prazo máximo de entrega em dias úteis informado no momento do cálculo do frete.

`discount`: (Float) O valor do desconto aplicado ao frete, se houver (ex: 7.43).

`height`: (String) A altura da embalagem em centímetros (ex: "15").

`width`: (String) A largura da embalagem em centímetros (ex: "25").

`length`: (String) O comprimento da embalagem em centímetros (ex: "26").

`weight`: (String) O peso da embalagem em quilogramas (ex: "2.20").

`from`: (Objeto) Detalhes do remetente:

* `address`: (String) Endereço do remetente.
* `city`: (String) Cidade do remetente.
* `complement`: (String) Complemento do endereço do remetente.
* `email`: (String) E-mail do remetente.
* `district`: (String) Bairro do remetente.
* `document`: (String) Documento do remetente (CPF/CNPJ).
* `name`: (String) Nome completo do remetente.
* `location_number`: (String) Número do endereço do remetente.
* `postal_code`: (String) CEP do remetente.
* `state_abbr`: (String) Código do estado (UF) do remetente.
* `country_id`: (String) Código do país do remetente (sempre "BR" para Brasil).

`to`: (Objeto) Detalhes do destinatário:

* `address`: (String) Endereço do destinatário.
* `city`: (String) Cidade do destinatário.
* `complement`: (String) Complemento do endereço do destinatário.
* `district`: (String) Bairro do destinatário.
* `document`: (String) Documento do destinatário (CPF/CNPJ).
* `name`: (String) Nome completo do destinatário.
* `location_number`: (String) Número do endereço do destinatário.
* `postal_code`: (String) CEP do destinatário.
* `state_abbr`: (String) Código do estado (UF) do destinatário.
* `country_id`: (String) Código do país do destinatário (sempre "BR" para Brasil).

`invoice`: (Objeto ou null) Detalhes da nota fiscal, se utilizada. Será null caso uma declaração de conteúdo seja utilizada.

`own_hand`: (Boolean) Indica se o serviço de Mão Própria foi solicitado (true) ou não (false).

`receipt`: (Boolean) Indica se o serviço de Aviso de Recebimento foi solicitado (true) ou não (false).

`price`: (Float) O valor total do frete (ex: 27.37).

`tracking`: (String ou null) O código de rastreamento da encomenda. Este campo só será preenchido após a etiqueta ser paga e o status mudar para **released** (aguardando postagem) ou posterior (ex: "AA923452383BR"). Será null se a etiqueta ainda não foi paga ou liberada.

`status`: (String) O status atual da etiqueta (ex: "delivered"). Os possíveis status incluem: pending, released, posted, delivered, canceled.

`service_id`: (Integer) O código do serviço de entrega utilizado (1 para PAC, 2 para SEDEX, 17 para Mini Envios).

`products`: (Array de Objetos) Lista dos produtos declarados para a encomenda (declaração de conteúdo):

* `name`: (String) Nome do produto.
* `quantity`: (String) Quantidade do produto.
* `unitary_value`: (String) Valor unitário do produto.

`insurance_value`: (Float ou null) O valor declarado do seguro da encomenda, se aplicável. Será null se nenhum seguro foi contratado.

`generated_at`: (String) Timestamp (em formato ISO 8601 UTC) da data e hora em que a etiqueta foi gerada.

`posted_at`: (String ou null) Timestamp (em formato ISO 8601 UTC) da data e hora em que a encomenda foi postada. Será null até a postagem ser registrada.

`created_at`: (String) Timestamp (em formato ISO 8601 UTC) da data e hora em que o pedido da etiqueta foi criado.

`updated_at`: (String) Timestamp (em formato ISO 8601 UTC) da última vez em que as informações da etiqueta foram atualizadas.

`print`: (Objeto) Informações para impressão da etiqueta:

* `url`: (String) URL para acessar o arquivo PDF da etiqueta para impressão.

`tags`: (Array de Objetos) Lista de tags associadas à etiqueta, geralmente contendo informações de identificação do pedido na plataforma de origem:

* `tag`: (String) O valor da tag (ex: "3101481").
* `url`: (String) A URL associada à tag na plataforma de origem
